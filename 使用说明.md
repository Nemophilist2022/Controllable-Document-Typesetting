# 论文排版工具 — 使用说明

## 两个程序文件的区别

在 `dist` 文件夹中有两个 `.exe` 文件：

| 文件 | 用途 |
|---|---|
| **thesis-format.exe** | 主程序。双击打开即可使用，所有排版功能都在这里面。 |
| **pandoc.exe** | 辅助转换工具。只有当你的论文原稿是 `.txt`、`.md` 或 `.tex` 格式时才需要。如果你的论文本身就是 `.docx` 或 `.doc`，可以忽略它。 |

> 两个文件必须放在同一个文件夹里，不要分开。

---

## 功能说明

双击 `thesis-format.exe` 打开后，左侧是 8 个功能分类，点击切换；右侧显示对应的设置项。底部是文件选择和执行区域。

所有设置项在打开时已填好华南农业大学 2024 本科论文的默认值，**大多数情况下不需要改动**，直接选文件、点"开始格式化"即可。

---

### 一、页面

设置纸张的页边距和前置页处理方式。

| 选项 | 说明 |
|---|---|
| 上/下/左/右边距 | 纸张四周的空白区域大小，单位厘米。 |
| 装订线 | 左侧额外留出的装订空间，用于打印后装订。 |
| 页眉距 / 页脚距 | 页眉、页脚到纸张顶部/底部的距离。 |
| 前置页处理 | 控制封面、声明页、摘要等前置内容的处理方式。三个选项：**自动识别并格式化**（推荐，程序自动找到前置页并排版）、**跳过**（完全保留前置页的原始格式不动）、**强制格式化**（不管是否识别准确，都按正文规则格式化前置页）。 |

---

### 二、封面声明

设置封面页和声明页的内容。

**封面部分：**

| 选项 | 说明 |
|---|---|
| 启用封面 | 勾选后程序会自动生成一页封面。取消勾选则不生成。 |
| 自定义封面 | 如果你已经有排好版的封面（`.docx` 文件），点"浏览"选择它，程序会直接用你的封面替代自动生成的。 |
| 学校名称 | 显示在日志和工具标题中的学校名。 |
| Logo 文件 | 封面上的校徽图片。默认已配好华农校徽，一般不用改。 |
| 封面标题 | 封面大标题文字。毕业设计请改为"本科毕业设计"。 |
| 信息栏字段 | 封面上"学院""专业""姓名""学号"等填写栏。可以添加或删除行，也可以修改每行的标签文字和下划线宽度。 |

**声明部分：**

| 选项 | 说明 |
|---|---|
| 启用声明页 | 勾选后会在封面之后插入原创性声明和授权声明两页。 |
| 声明标题/正文 | 可以直接修改声明的标题和正文内容。默认已填入华农标准文本。 |

---

### 三、正文

设置正文的字体、字号、段落格式。

| 选项 | 说明 |
|---|---|
| 西文字体 | 英文字母和数字使用的字体。 |
| 正文中文字体 | 中文正文使用的字体。 |
| 正文字号 | 正文文字大小。默认 12pt（即"小四"）。 |
| 正文对齐 | 段落对齐方式。"两端对齐"是最常用的。 |
| 首行缩进 | 每段第一行向右缩进的距离。24pt 等于两个中文字符宽度。 |
| 行距 | 行与行之间的间距。常见选择有"单倍""1.5 倍""2 倍"。选择"固定值"或"最小值"时，需要在右侧手动输入数值。 |
| 段前 / 段后 | 每个段落上方/下方额外的间距。0 表示没有额外间距。 |
| 脚注字号 | 页面底部脚注文字的大小。 |
| 脚注行距 | 脚注内部的行间距。 |

---

### 四、标题

设置一级到四级标题的样式和编号规则。

**各级标题样式（一级 ~ 四级）：**

| 选项 | 说明 |
|---|---|
| 字体 | 该级别标题使用的中文字体。 |
| 字号 | 标题文字大小。 |
| 加粗 | 是否加粗。"保持原样"表示不修改原文档中的粗体设置。 |
| 对齐 | 标题的对齐方式。 |
| 段前 / 段后 | 标题上下方的额外间距。填 -1 表示保持原样不修改。 |

**编号设置：**

| 选项 | 说明 |
|---|---|
| 自动修正标题编号 | 勾选后，程序会检查标题编号是否连续（比如 1、2、3 有没有跳号），并自动修正。 |
| 编号预设 | 选择标题编号风格。华农理工农科类常用"X / X.X / X.X.X（纯数字）"，即一级标题用"1"，二级用"1.1"，三级用"1.1.1"。选择预设后会自动填入下方的识别模式。 |
| 一级~四级标题识别模式 | 程序用来识别哪些段落是标题的规则（正则表达式）。选了编号预设后会自动填好，**一般不需要手动修改**。 |

---

### 五、页眉页码

设置页眉和页码的格式。

**页眉部分：**

| 选项 | 说明 |
|---|---|
| 启用页眉 | 是否添加页眉。华农本科论文规范不要求页眉，默认关闭。 |
| 作用范围 | 页眉出现在哪些页面。"仅正文"表示封面和摘要页不加页眉。 |
| 奇偶页不同 | 勾选后，可以分别设置奇数页和偶数页的页眉内容。 |
| 首页不显示页眉 | 勾选后，每一节的第一页不显示页眉。 |
| 奇数页/偶数页文字 | 页眉显示的文字。勾选"自动显示章标题"后，页眉会自动跟随当前章的标题文字。 |
| 页眉字体/字号/加粗 | 页眉文字的样式。 |
| 页眉下划线 | 勾选后在页眉下方添加一条横线。可设置线条粗细和样式（单线/双线）。 |

**页码部分：**

| 选项 | 说明 |
|---|---|
| 前置页码位置 / 正文页码位置 | 页码在页面底部的位置。"居中"最常用。"奇右偶左"表示奇数页页码靠右、偶数页靠左。 |
| 前置页码格式 / 正文页码格式 | 页码的数字格式。前置部分（摘要、目录等）通常用罗马数字，正文用阿拉伯数字。 |
| 前置/正文起始编号 | 页码从几开始计数。 |
| 页码修饰 | 页码两侧的装饰文字。`page}` 代表页码数字本身，例如填 `- {page} -` 会显示为 `- 1 -`。 |
| 页码字体/字号/加粗 | 页码数字的样式。字体留空则跟随西文字体设置。 |

---

### 六、目录参考

设置目录样式、参考文献格式和特殊标题。

**目录部分：**

| 选项 | 说明 |
|---|---|
| 目录深度 | 目录显示到第几级标题。3 表示显示一级、二级、三级标题。 |
| 一级条目字体/字号/加粗 | 目录中一级标题条目的样式。 |
| 二级条目字体/字号/加粗 | 目录中二级及以下标题条目的样式。 |
| 目录行距 | 目录条目之间的行间距。 |
| 摘要/Abstract 不参与目录 | 勾选后，中英文摘要的标题不会出现在目录中。 |
| 条目段前 / 段后 | 每条目录条目上下方的间距。 |

**参考文献部分：**

| 选项 | 说明 |
|---|---|
| 参考文献缩进 | 参考文献列表的缩进量。程序会将参考文献设置为悬挂缩进格式（第一行顶格，后续行缩进），此值控制缩进大小。 |

**特殊标题映射：**

控制"摘要""目录""参考文献""致谢"等不带编号的特殊标题在正文中的显示方式。

| 列 | 说明 |
|---|---|
| 匹配 | 原文中用来识别的关键词。程序会忽略空格来匹配，所以不管原文写的是"参考文献"还是"参 考 文 献"，都能识别到。 |
| 显示 | 格式化后在正文中实际显示的文字。可以用全角空格控制字间距，例如"致　　　　谢"会让"致"和"谢"之间有较大间隔。 |
| 对齐 | 该标题的对齐方式。 |

---

### 七、图表

设置图表题注的字体、识别规则和三线表格式。

**题注部分：**

| 选项 | 说明 |
|---|---|
| 图表题字号 | "图 1 XXX""表 1 XXX"这类题注文字的大小。 |
| 题注行距 | 题注的行间距。 |
| 题注字体 | 题注中文字使用的字体。 |
| 编号字体 | 题注中数字使用的字体。 |
| 题注模式 | 有两种模式：**严格动态模式**会使用 Word 的自动编号功能，插入新图表后编号自动更新；**稳定模式**保持原文中的编号文字不变。推荐使用动态模式，如果文档结构不规范，程序会自动回退到稳定模式。 |
| 图表题防分页 | 勾选后，图表题注会与它对应的图/表保持在同一页，不会被分页拆开。 |
| 检查图表编号连续性 | 勾选后，程序会检查图表编号是否连续（如图 1、图 2、图 3 有没有跳号），不连续时给出警告。 |
| 图表按章编号 | 勾选后，图表编号会带章号，如"图 2.1"表示第 2 章第 1 张图。华农规范要求全文连续编号（图 1、图 2...），所以默认关闭。 |
| 每章重新编号 | 勾选后，每一章的图表编号从 1 重新开始。需要配合"图表按章编号"使用。 |
| 章节标题级别 | 哪一级标题算作"章"。通常选 1（一级标题）。 |
| 章号分隔符 | 章号和图表序号之间的符号，如"."会显示为"图 2.1"。 |
| 编号后分隔符 | 编号和题注文字之间的附加符号。默认留空。 |

**题注识别模式：**

程序用来识别原文中哪些段落是图题、表题的规则。默认值能识别"图 1""表 1""(a)""注："等常见格式，**一般不需要修改**。

**三线表部分：**

| 选项 | 说明 |
|---|---|
| 顶线/栏目线/底线粗细 | 三线表三条横线的粗细。学术论文常用顶线和底线 1.5 磅、栏目线 1.0 磅。 |
| 表格行距 | 表格内部文字的行间距。 |

---

### 八、单独设置

当你不想做全套排版，只想单独执行某一个操作时，在这里勾选。

| 选项 | 说明 |
|---|---|
| 单独插入/更新目录 | 只生成或刷新目录，不修改正文和其他格式。适合手动排版后只需要加目录的情况。 |
| 单独插入外部封面 | 只把你准备好的封面 `.docx` 插入到论文开头，其他内容不动。使用前需要先在"封面声明"里选好封面文件。 |
| 单独改页码 | 只调整页码格式和编号，不动正文内容。按文档中现有的分节符来设置页码。 |
| 单独改页眉 | 只调整页眉内容和格式，不动正文。按现有分节符来设置页眉。 |

> 这四个选项互斥，同一时间只能勾选一个。

---

### 底部区域

| 选项 | 说明 |
|---|---|
| 恢复默认 | 把所有设置恢复到华农 2024 的默认值。 |
| 保存配置 | 把当前所有设置保存成一个 `.yaml` 文件，下次可以直接加载。 |
| 加载配置 | 加载之前保存的配置文件，自动填入所有设置。 |
| 输入文件 | 选择你要排版的论文文件。支持 `.docx`、`.doc`、`.txt`、`.md`、`.tex` 格式。 |
| 输出文件 | 排版后的文件保存位置。选择输入文件后会自动生成，也可以手动修改。 |
| 跳过目录生成 | 勾选后不生成目录。适合论文还没写完、不需要目录的阶段。 |
| 开始格式化 | 点击后开始执行排版。执行过程中按钮会变灰，完成后会显示结果。 |
| 运行日志 | 显示执行过程中的详细信息，包括进度、警告和错误。如果格式化出了问题，可以在这里查看原因。 |

## 操作流程

1. **双击** `thesis-format.exe` 打开工具。
2. **检查设置**（可选）：左侧 8 个分类里的默认值已按华农规范配好，通常不需要改动。如有特殊需求再逐项调整。
3. **选择文件**：点击底部"浏览"选择你的论文文件，输出路径会自动生成。
4. **点击"开始格式化"**：等待进度条走完，日志区域显示"格式化完成"即可。
5. **打开输出文件**：用 Word 打开生成的 `_formatted.docx` 文件，右键目录区域 → 更新域 → 更新整个目录。

---

## 建议

- **先备份原文件**。虽然程序不会修改你的原始文件（它会生成一个新的 `_formatted.docx`），但养成备份习惯总没错。
- **使用 `.docx` 格式**作为输入。`.doc` 格式需要电脑上安装了 Word 才能转换，`.txt`/`.md`/`.tex` 需要 `pandoc.exe`。直接用 `.docx` 最省事。
- **论文写完再排版**。排版工具对已完成的论文效果最好。如果论文还在修改中，后续改动可能需要重新格式化。
- **检查图表题注格式**。程序靠"图 1""表 1"这样的文字开头来识别题注，请确保你的图表题注符合这个格式。
- **排版后在 Word 中更新目录**。程序会自动更新目录，但如果看到目录显示为提示文字，在 Word 中右键目录 → 更新域 → 更新整个目录即可。
- **遇到问题看日志**。底部"运行日志"会详细记录每一步的执行情况和警告信息，是排查问题的第一手资料。
